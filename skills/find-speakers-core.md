# Skill: find-speakers-core

## When to use
Когда пользователь вводит команду `/find_speakers <сезон> <регион>` в боте parsing_speakers_kant_v2.

## Context
Бот ищет потенциальных спикеров для лекций и мероприятий сети KANT по зимним и летним видам спорта.

## Input
- season: "зима" или "лето"
- location_scope: "Екатеринбург", "Свердловская область", "УрФО", "Россия"
- sport_focus (optional): если пользователь уточняет вид спорта вручную.

## Instructions
1. Найди 3–7 релевантных спикеров по заданному сезону и региону.
2. Для каждого спикера опиши:
   - name
   - sport
   - expertise (1–2 предложения)
   - city/region
   - url (Instagram, VK, сайт или профиль мероприятия — если нет, оставь пустую строку).
3. Если спикеров нет, верни пустой массив `speakers: []`, но обязательно укажи нормальный `season` и `location_scope`.

## Output format (JSON)
Возвращай только корректный JSON без пояснений:

{
  "season": "<зима|лето>",
  "location_scope": "<Екатеринбург|Свердловская область|УрФО|Россия>",
  "speakers": [
    {
      "name": "Имя Фамилия",
      "sport": "конкретный вид спорта",
      "expertise": "краткое описание экспертизы",
      "city": "город",
      "url": "https://..."
    }
  ]
}
