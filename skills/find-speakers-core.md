# Skill: find-speakers-core

## Purpose
Базовый навык поиска спикеров для лекций/мероприятий KANT по сезону (зима/лето) и региону.

## When to use
Используй этот навык, когда пользователь вводит команду формата:
/find_speakers <сезон> <регион>

Примеры:
- `/find_speakers зима Екатеринбург`
- `/find_speakers лето УрФО`
- `/find_speakers лето Россия`

## Input
- season: "зима" или "лето"
- location_scope: один из вариантов:
  - "Екатеринбург"
  - "Свердловская область"
  - "УрФО"
  - "Россия"
- sport_focus (опционально): конкретный вид спорта, если пользователь его явно указал.

## Instructions
1. Понимай задачу как подбор спикеров для лекций, мастер-классов и мероприятий сети магазинов KANT.
2. Найди 3–7 релевантных спикеров с учётом:
   - выбранного сезона,
   - выбранного региона/радиуса,
   - указанного вида спорта (если есть).
3. Для каждого спикера укажи:
   - name — имя и фамилия;
   - sport — основной вид спорта или специализация;
   - expertise — 1–2 предложения о темах лекций и опыте;
   - city — город спикера;
   - url — ссылка на профиль/сайт/страницу мероприятия (если нет, оставь пустую строку "" без выдумываний).
4. Если подходящих спикеров мало, можешь вернуть 2–3, но не нарушай формат.
5. Если спикеров нет, верни корректные поля season и location_scope, а `speakers` сделай пустым массивом.

## Output format
Отвечай только валидным JSON без пояснений вокруг.

Пример структуры:

{
  "season": "зима",
  "location_scope": "Екатеринбург",
  "speakers": [
    {
      "name": "Имя Фамилия",
      "sport": "беговые лыжи",
      "expertise": "Тренер по беговым лыжам, проводит лекции для любителей и подготовку к соревнованиям.",
      "city": "Екатеринбург",
      "url": "https://..."
    }
  ]
}
